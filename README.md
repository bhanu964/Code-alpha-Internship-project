
# 🧠 Credit Scoring Model

This project predicts an individual's creditworthiness using machine learning classification algorithms like Logistic Regression, Decision Tree, and Random Forest.

## 🔍 Objective
To classify individuals as **Good** or **Bad** credit risks based on financial history.

## 📁 Dataset Features
- Age
- Income
- Employment Status
- Credit Score
- Loan Amount
- Payment History
- Defaulted Before

## ⚙️ Tech Stack
- Python
- scikit-learn
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## 📊 Evaluation Metrics
- Accuracy
- Precision, Recall, F1-Score
- ROC-AUC Score
- Feature Importance

## 📈 Output
- Model performance comparison
- ROC curves
- Feature importance charts

## 🚀 How to Run
1. Clone the repo
```bash
git clone https://github.com/your-username/credit-scoring-model.git
cd credit-scoring-model
```
2. Create a virtual environment and install requirements
```bash
pip install -r requirements.txt
```
3. Open the Jupyter notebook
```bash
jupyter notebook notebooks/credit_model.ipynb
```
